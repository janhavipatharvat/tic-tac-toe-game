# PRODIGY_WD_03

## Tic Tac Toe Web Application

This project is developed as part of my Web Development Internship at Prodigy Infotech.

It is an interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

---

## 📌 Features

- 3x3 interactive game board
- Player vs Player functionality
- Turn indicator display
- Automatic win detection
- Winner cell highlight
- Draw detection
- Restart game option
- Dark / Light mode toggle

---

## 🛠 Technologies Used

- HTML (Structure)
- CSS (Styling & Grid Layout)
- JavaScript (Game Logic & DOM Manipulation)
